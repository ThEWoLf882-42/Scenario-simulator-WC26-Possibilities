# WC26 Possibilities

A standalone FIFA World Cup 2026 scenario simulator.

## Open it

### Fastest

Double-click `index.html`.

### Recommended local server

On Windows, double-click `start-server.bat`, then open:

`http://localhost:8080`

## Included

- All 12 groups and 72 group-stage matches.
- Real results preloaded through 23 June 2026.
- Editable scores with instant standings.
- Head-to-head-first group ranking, then overall goal difference and goals scored.
- Best eight third-place cutline.
- Round-of-32 projection using FIFA’s exact 495-combination Annex C third-place matrix.
- Morocco-focused qualification and opponent view.
- English/Arabic switch.
- Dark/light mode.
- Save, export and import scenarios.

## Important ranking note

Exact ties after points, relevant head-to-head values, goal difference and goals scored may require official team-conduct and FIFA-ranking criteria. The app flags the practical score-based projection, while FIFA remains the official source.

## Data snapshot

The included real-result snapshot is current through matches completed on 23 June 2026. Future results are intentionally editable.

## World Cup 26 theme

The interface uses the included official World Cup 26 emblem, a black-and-white editorial base, tournament color accents, poster-style cards, and a matching two-sided knockout bracket. The theme supports desktop, mobile, English, Arabic, dark mode, and light mode.
